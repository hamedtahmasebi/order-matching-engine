# Order matching engine in GO

This is a simple order matching engine implemented in Go. It supports matching orders for different pairs of assets.

This is my attempt to implement an order matching engine in Go. 
It is a simple implementation that supports matching orders for different pairs of assets. 
The purpose of this project is for me to learn more about Go and to gain experience in building an application with real-world use-case.


### What is an order matching engine?
An order matching engine is a system that matches buy and sell orders for a particular asset or security. 
It is used in financial markets to facilitate trading and ensure that orders are executed at the best possible price.
